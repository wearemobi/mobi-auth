A cloud-native Identity &amp; Access Management (IAM) server. Built with Java 21, Spring Boot 3.4, and Spring Authorization Server

Mobi Auth 🛡️ is the centralized security backbone of the M.O.B.I. ecosystem. It provides a robust, standards-compliant Authorization Server.

🚀 Features
- Standards-Based: Full implementation of OAuth 2.1 and OpenID Connect 1.0.
- Modern Stack: Leveraging the power of Java 21 (Virtual Threads ready) and Spring Boot 3.4.
- Identity Federation: Designed for seamless SSO across *.mobi.bio subdomains.
- Security First: Implements PKCE, JWT-based claims, and secure credential hashing.

🛠️ Tech Stack
- Language: Java 21
- Framework: Spring Boot / Spring Security / Spring Authorization Server
- Database: HSQL/MySQL/PostgreSQL/Oracle (Spring Data Persistence layer)
- API Standards: OAuth2, OIDC, JWT
